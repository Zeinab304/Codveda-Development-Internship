# C/C++ Development Internship - Codveda 🚀

Welcome to my repository for the C/C++ Development Internship at **Codveda Technology**! This repository contains the implementations for the internship tasks, developed using both **C++** and **C** languages.

## 🏢 About the Internship
This internship with **Codveda** focuses on mastering foundational and advanced concepts in software development, ranging from basic logic building to complex modern software engineering concepts like systems simulation and multithreading.

## Tasks List

### Level 1: Basic
* Task 1: Basic Calculator
- Description: A console-based calculator performing fundamental arithmetic operations while handling edge-cases like division by zero.
- Skills: Input/Output, Conditional Logic.

* Task 2: Number Guessing Game
- Description: An interactive game generating dynamic random numbers using rand() where the user guesses the target with high/low hints.
- Skills: Random number generation, Loops, Conditional logic.

* Task 3: String Manipulation
- Description: A program implementing basic string operations such as reversing a string, counting vowels, and checking for palindromes.
- Skills: Character arrays, Custom functions, String operations.

### Level 2: Intermediate
* Task 1: Simple Student Management System
- Description: A complete console CRUD application utilizing structures and file handling to manage student records.
- Skills: Structs, File I/O, Arrays, Pointers.

* Task 2: Sorting and Searching Algorithms
- Description: Implementation of popular sorting (Bubble sort, Quicksort) and searching (Linear, Binary) algorithms with time complexity comparison.
- Skills: Algorithms, Time complexity analysis, Arrays.

* Task 3: Basic Data Structure Implementation
- Description: Implementing fundamental data structures like Stacks or Queues from scratch using arrays or linked lists.
- Skills: Data structures, Dynamic memory allocation, Linked lists.

### Level 3: Advanced
* Task 1: Simple File System Simulation
- Description: A simulation of a file system architecture to perform core operations like creating, reading, and deleting files.
- Skills: Advanced File I/O, Structs, Pointers, Error Handling.

* Task 2: Multithreaded Application
- Description: A multithreaded application simulating the classic Producer-Consumer problem using POSIX threads.
- Skills: Thread synchronization, Mutexes, Condition variables.

* Task 3: Mini Game Development (Tic-Tac-Toe)
- Description: A classic console-based Tic-Tac-Toe game for two players, handling turn switching and win/draw conditions.
- Skills: Game logic, Array grid management, Turn-based input validation.
